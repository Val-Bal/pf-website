IndieFlix (server)

REST API for a movie app called IndieFlix that interacts with a database that stores data for a certain catalogue of movies. The app provides users with a list of movies and their respective plot summary, information about directors and genres. The users are able to sign up to/delete their account, update their user information and add/remove their favorite movies to a list in their user profile.

Technologies used: JavaScript, MongoDB, Express, Node.js, Postman, PostgreSQL, Mongoose
